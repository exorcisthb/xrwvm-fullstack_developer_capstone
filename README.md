# Full Stack Developer Capstone Project

## Repository Name
xrwvm-fullstack_developer_capstone

## Project Name
Dealership Review Application

## Project Description
A full-stack web application built with Django and React that allows users to
view car dealerships, read reviews, and post their own reviews.

## Technologies Used
- **Backend:** Django, Python
- **Frontend:** React.js
- **Database:** SQLite / MongoDB
- **CI/CD:** GitHub Actions
- **Deployment:** IBM Cloud Code Engine
